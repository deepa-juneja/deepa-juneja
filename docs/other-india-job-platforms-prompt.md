# Other India Job Platforms — Apna / Foundit / Hirect / iimjobs / TimesJobs / Cutshort

Use this prompt for sessions on the smaller / specialised Indian job platforms. Each has a distinct hiring pool and UX. Do **not** treat them with the same priority as Naukri, LinkedIn, Indeed, or Google for Jobs — they're a complement, not a substitute.

```text
You are helping search and apply for jobs on smaller Indian job platforms
for Deepa Juneja. Use this after the main three (Naukri, LinkedIn, Indeed)
have been worked. Each platform below has unique listings worth a 15-30
minute pass per session.

Candidate profile, role buckets, role-to-resume mapping, salary positioning,
screening answers, cover letter template, and avoid list:
**Use the definitive copies in:**
  docs/naukri-job-search-agent-prompt.md
Apply the same standards here.

========================================================================
## 1. Apna (apna.co)
========================================================================
Mobile-first job platform. Strongest for entry / mid roles in BPO, ops,
sales, customer support, and field jobs in Gurgaon/Delhi NCR. Many roles
allow direct WhatsApp/call connection with the HR.

URL: https://apna.co/jobs
Web works, but the mobile app is faster (open on phone if available).

What's good here:
- Very fast callback loop — many HRs message within hours
- Profile-only applying (no resume needed for ~50% of listings)
- Some listings allow custom resume upload — use it for KPO MIS,
  EdTech, training, HR coordination roles
- "Verified" employer tier — prefer these; the unverified ones are
  hit-or-miss
- Direct WhatsApp / call icon on many listings — bypasses the apply queue

What's weak here:
- Listings skew toward freshers, BPO voice, field sales — Deepa is
  overqualified for many
- Lots of agency / consultancy postings — quality varies
- Salaries are often understated (recruiters negotiate up)

Search batches:
- "MIS Executive" — Gurgaon
- "Operations Executive" — Gurgaon
- "HR Coordinator" — Gurgaon
- "Recruitment Executive" — Gurgaon
- "Training Coordinator" — Gurgaon
- "Counsellor" — Gurgaon
- "Academic Coordinator" — Gurgaon
- "Customer Support Executive" — Gurgaon (KPO leaning, decent pay)
- "Backend Operations" — Gurgaon
- "Data Entry / MIS" — Gurgaon (filter for MIS, skip pure data-entry)

Apply behaviour:
1. Filter to "Verified" employers when possible.
2. Open the listing. Check stated pay range. Skip < 2 LPA outright;
   apply if 2-3 LPA only when callback probability looks high.
3. If "Apply" button → submit profile snapshot.
4. If "WhatsApp HR" or "Call HR" button → use these for high-fit roles.
   Send a 1-sentence WhatsApp: "Hi, I saw your <role> opening at <co>.
   12+ years across teaching, school coordination, and HR ops, MBA + MCA.
   Sharing my resume — happy to discuss." Attach the role-tailored PDF.
5. Skip clearly target-driven field sales unless pay > 4 LPA base.

Avoid on Apna:
- Field sales (door-to-door, micro-loans, insurance cold-calling)
- Listings with vague pay like "₹15k-50k/month" and no JD
- "Earn from home" / "Data entry" scams — these often appear; ignore
========================================================================

## 2. Foundit (foundit.in, formerly Monster India)
========================================================================
Older platform, declining traffic but some employers post here exclusively.
Similar UX to Naukri.

URL: https://www.foundit.in/srp/results
Same search → filter → apply pattern as Naukri.

What's good here:
- Some mid-size and older Indian companies still post here only
- Decent for KPO MIS, training, school coordinator roles
- Lower applicant counts than Naukri (less competition per role)

What's weak here:
- Total listing volume is much lower than Naukri
- Some listings are stale / re-posted from years ago
- UX is dated

Search batches (run same as Naukri primary buckets):
- "MIS Executive" — Gurgaon
- "Training Coordinator" — Gurgaon
- "Academic Coordinator" — Gurgaon
- "HR Coordinator" — Gurgaon
- "Operations Executive" — Gurgaon
- "School Coordinator" — Gurgaon

Apply behaviour: same as Naukri (profile-based + per-application resume
upload available for many roles). Pick role-tailored PDF when offered.

## 3. Hirect (hirect.in)
========================================================================
Chat-based hiring platform. Direct chat with the founder/HR/hiring
manager. Strongest for startup / SaaS / mid-sized tech-adjacent roles.

URL: https://www.hirect.in/
Mobile app is primary; web available.

What's good here:
- No traditional apply form — you chat directly with the hiring side
- Fast turnaround (response within hours typical)
- Good for ops/training roles at SaaS startups (Razorpay, Cleartax,
  Zoho, Freshworks-adjacent, smaller B2B SaaS)
- Lower competition than Naukri for similar listings

What's weak here:
- Roles skew toward Bangalore — fewer Gurgaon listings
- Smaller listing pool overall
- Some chats go cold quickly if response time is slow

Search batches:
- "Operations" — Gurgaon, Remote
- "Customer Success" — Gurgaon, Remote
- "HR Coordinator" — Gurgaon, Remote
- "L&D" — Gurgaon, Remote
- "MIS Analyst" — Gurgaon, Remote

Apply behaviour:
1. Initiate chat with a 200-char intro:
   "Hi — saw the <role> posting. 12+ years across teaching, school
   coordination, MBA + MCA, recent HR internship at GAO Tek, currently
   on Google AI Essentials. Portfolio: deepa-juneja.github.io/deepa-juneja.
   Happy to share resume and discuss fit."
2. Attach the role-tailored PDF in the chat (Hirect supports file
   attachments in chat).
3. Don't chat with 20 people in 5 minutes — pace one every 5-10 minutes
   so responses don't pile up faster than you can handle them.
4. If no response in 48 hours, follow up once with a single line; then
   move on.

## 4. iimjobs.com / hirist.com (one company, two brands)
========================================================================
Premium mid-senior platform. Roles 5+ years, ₹6 LPA+. iimjobs is for
non-tech, hirist is for tech. Deepa's profile fits iimjobs better.

URL: https://www.iimjobs.com/

What's good here:
- Higher salary band (₹6-25 LPA range is common)
- Lower competition (premium positioning means fewer applicants)
- Many roles in HR, Operations, L&D, Training that match Deepa's stretch
- Some EdTech and corporate L&D roles posted exclusively here

What's weak here:
- Many roles ask for 5-10 years in a specific function — Deepa's
  experience is distributed across teaching/coordination, which some
  employers may discount
- Apply system can be slow / requires creating a profile first
- Some roles charge nothing; some are paid-applicant only — check
  before investing time

Search batches:
- "Training & L&D" — Gurgaon
- "HR Operations" — Gurgaon
- "EdTech Operations" — Gurgaon
- "Programme Manager Education" — Gurgaon
- "Academic Operations Manager" — Gurgaon
- "L&D Specialist" — India

Apply behaviour:
- Use the role-tailored PDF (Training & L&D or HR Operations).
- Cover letter is often a free-text field — paste the cover letter
  template from the Naukri prompt and personalise per role.
- These applications are slower to process; expect 1-2 week response
  cycles. Don't chase if no response by day 14.

## 6. Cutshort (cutshort.io)
========================================================================
Tech-skewed but has Ops, HR, and L&D roles at startups. Smaller pool
for non-tech profiles, but worth a quick check.

URL: https://cutshort.io/

What's good here:
- Verified employer-side workflow — fewer junk listings
- Direct chat with hiring side
- Good for ops/HR at funded startups

What's weak here:
- Most roles want product/engineering/data background — not Deepa's fit
- Limited Gurgaon focus (Bangalore-heavy)

Search batches (quick pass):
- "Operations" — Gurgaon, Remote
- "HR" — Gurgaon, Remote
- "L&D" — Gurgaon, Remote
========================================================================

Overall strategy for this prompt:
- Spend at most 30-45 minutes per platform per session.
- Apna is highest-volume — go first if running short on time.
- Foundit and TimesJobs are quick passes — 10 min each.
- Hirect and Cutshort are chat-paced — limit to 5-10 conversations to
  avoid response overload.
- iimjobs is for premium stretch applications — quality over quantity.

Output format after a multi-platform session:
- Per platform, list:
  ✅ Applied — Company | Role | Resume variant | Pay quoted | Posted
  ⚠ Blocked / chat-gated
  💬 Chat-initiated (Hirect / Cutshort) — pending response
  🚩 Flagged (suspicious / vague)
- Overall:
  📈 Session totals across platforms
  🎯 Best 3-5 follow-up actions for next time
```
