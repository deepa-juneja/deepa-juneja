# Documentation

## Job-search agent prompts

Copy any of these prompts into a fresh Claude session to drive a job-search workflow on the target platform. Each prompt is **self-contained** for the operating instructions, but cross-references `naukri-job-search-agent-prompt.md` for the **shared candidate profile, role buckets, role-to-resume mapping, salary positioning, screening answers, and cover letter template** — keep that file open in the same conversation when you run one of the others.

| Platform | Prompt file | Strength | Use when |
|---|---|---|---|
| **Naukri** | [`naukri-job-search-agent-prompt.md`](naukri-job-search-agent-prompt.md) | Highest volume of Indian listings; broadest coverage | Default first stop. Run weekly. |
| **LinkedIn Jobs** | [`linkedin-jobs-search-agent-prompt.md`](linkedin-jobs-search-agent-prompt.md) | Best signal quality; corporate/EdTech/L&D; DM-the-poster advantage | After Naukri. Great for high-value applications + warm intros. |
| **Google for Jobs** | [`google-jobs-search-agent-prompt.md`](google-jobs-search-agent-prompt.md) | Meta-search across all platforms; surfaces company-website-only roles | Discovery layer. Run weekly to find listings missed elsewhere. |
| **Indeed India** | [`indeed-jobs-search-agent-prompt.md`](indeed-jobs-search-agent-prompt.md) | Unique pool of small-to-mid Indian companies, KPO/BPO ops, schools | Third stop after Naukri + LinkedIn. |
| **Apna / Foundit / Hirect / iimjobs / TimesJobs / Cutshort** | [`other-india-job-platforms-prompt.md`](other-india-job-platforms-prompt.md) | Niche platforms with unique listings; chat-based options | Spot pass — 15-30 min per platform. |

### Recommended weekly cadence

1. **Mon** — Naukri (60–90 min, target 20–40 applications)
2. **Tue** — LinkedIn Jobs (45–60 min Easy Apply + 5–10 DMs to posters)
3. **Wed** — Google for Jobs discovery (30 min) → route findings to Naukri/LinkedIn/Indeed
4. **Thu** — Indeed India (45 min)
5. **Fri** — Apna + one other niche platform from `other-india-job-platforms-prompt.md`
